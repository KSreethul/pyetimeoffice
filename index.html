pyetimeoffice ETimeOffice Python Integration

Python integration library for ETimeOffice biometric attendance systems, attendance APIs, Django HRMS integrations, employee punch synchronization, and biometric attendance workflows.

GitHub Repository PyPI Package

Features

Attendance APIs

Download punch data, MCID records, and in/out attendance logs from ETimeOffice systems.

Django HRMS Integration

Easily integrate biometric attendance workflows into Django HRMS and ERP platforms.

Biometric Synchronization

Automate employee attendance synchronization and attendance tracking workflows.

Python Friendly

Lightweight Python API client with clean methods and normalized date handling.

Installation

pip install pyetimeoffice
